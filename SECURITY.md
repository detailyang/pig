# Security Policy

Pig is an early open-source library. Please report suspected vulnerabilities privately instead of opening a public issue.

## Reporting

Email the maintainer or use a private GitHub security advisory if the repository has advisories enabled.

Please include:

- Affected package or API surface.
- Steps to reproduce.
- Impact and any known mitigations.
- Whether the issue requires live provider credentials, local files, MCP servers, or shell tools.

## Secret Handling

Do not include API keys, tokens, private session transcripts, or SSH material in reports. Redact sensitive values before sharing logs.
